A free Firebase database is needed to sync game state between players. It takes 2 minutes to set up.

One-time setup:
  1. Go to console.firebase.google.com
  2. Create a project → Add Realtime Database
  3. Start in test mode (allows all reads/writes)
  4. Copy the database URL (looks like https://your-project-default-rtdb.firebaseio.com)
  5. Paste it below and click Save
Your Firebase URL is saved in your browser — you only need to do this once.
